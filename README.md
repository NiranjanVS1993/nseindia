**NSE India Automation**
This project is about automating the NSE India Website to get the information on the Market and Stocks for 52 weeks high and low data

**Main Features:**

Getting live quotes for stocks using stock codes
Getting quotes for all the indices traded in NSE, e.g RCOM,TATAMOTORS,INFOSYS, etc
Getting list of top losers
Getting list of top gainers
Helper methods to check whether a given stock code or index code is correct

**Getting Started**
